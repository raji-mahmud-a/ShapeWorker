ShapeWorker Documentation

A consolidated JavaScript library for working with geometric shapes. Provides methods for calculating area, perimeter, surface area, volume, and more across 2D and 3D shapes using a single class API.

Installation

Include via CDN or ES Module import:

CDN

<script src="https://mahmud-links.pxxl.xyz/shapeWorker.js"></script>

Import

import ShapeWorker from "./shapeWorker.js";

Instantiate new ShapeWorker("shape", ...params) with the shape name and parameters.

Examples

2D Shapes

Square

A quadrilateral with all sides equal.

const square = new ShapeWorker("square", 5);
square.area();       // 25
square.perimeter();  // 20
  • area() → Returns side × side
  • perimeter() → Returns 4 × side

Rectangle

A quadrilateral with opposite sides equal.

const rectangle = new ShapeWorker("rectangle", 4, 6);
rectangle.area();       // 24
rectangle.perimeter();  // 20
  • area() → Returns length × breadth
  • perimeter() → Returns 2 × (length + breadth)

Circle

A round plane figure defined by its radius.

const circle = new ShapeWorker("circle", 3);
circle.area();          // 28.274...
circle.circumference(); // 18.849...
  • area() → Returns π × r²
  • circumference() → Returns 2 × π × r

Triangle

A polygon with three sides. Supports base/height or Heron's formula.

const triangle1 = new ShapeWorker("triangle", 10, 6, 4);
triangle1.area(); // 12

const triangle2 = new ShapeWorker("triangle", 10, 6, "45deg");
triangle2.area(); // ~21.213
  • area() → Returns area based on given sides or angle

Parallelogram

A quadrilateral with opposite sides parallel.

const parallelogram = new ShapeWorker("parallelogram", 8, 5, "30deg");
parallelogram.area();      // 20
parallelogram.perimeter(); // 26
  • area() → Returns base × height or base × side × sin(angle)
  • perimeter() → Returns 2 × (base + side)

3D Shapes

Sphere

A perfectly round 3D object.

const sphere = new ShapeWorker("sphere", 5);
sphere.surfaceArea(); // 314.159...
sphere.volume();      // 523.598...
  • surfaceArea() → Returns 4 × π × r²
  • volume() → Returns (4/3) × π × r³

Cylinder

A solid with circular ends.

const cylinder = new ShapeWorker("cylinder", 3, 7);
cylinder.surfaceArea(); // 188.495...
cylinder.volume();      // 197.920...
  • surfaceArea() → Returns 2πr(h + r)
  • volume() → Returns πr²h

Cube

A solid with six equal square faces.

const cube = new ShapeWorker("cube", 4);
cube.surfaceArea(); // 96
cube.volume();      // 64
  • surfaceArea() → Returns 6 × side²
  • volume() → Returns side³

Cuboid

A rectangular solid.

const cuboid = new ShapeWorker("cuboid", 2, 3, 4);
cuboid.surfaceArea(); // 52
cuboid.volume();      // 24
  • surfaceArea() → Returns 2(lb + bh + hl)
  • volume() → Returns l × b × h

Cone

A solid with a circular base tapering to a point.

const cone = new ShapeWorker("cone", 3, 8);
cone.surfaceArea(); // 103.672...
cone.volume();      // 75.398...
  • surfaceArea() → Returns πr(l + r)
  • volume() → Returns (1/3)πr²h

Prism

Represents a prism defined by the area of its base (BaseArea) and height (h).

const prism = new ShapeWorker("prism", 20, 10);
prism.volume();      // 200
prism.surfaceArea ();   // placeholder message
  • surfaceArea() → Returns a placeholder (not yet implemented)
  • volume() → Calculates the volume of the prism as BaseArea × h
