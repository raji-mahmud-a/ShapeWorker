Shape Worker Documentation

A simple JavaScript library for working with geometric shapes. Provides methods for calculating area, perimeter, and more across common 2D shapes.

Installation

You can use shapeWorker directly in the browser via a CDN link. Simply add the following script tag inside your <head> or before the closing </body> tag:

CDN Usage


<script src="https://mahmud-links.pxxl.xyz/shapeWorker.js"></script>

After including the script, you can directly access all classes such as Square, Rectangle, Circle, Parallelogram, and Triangle.

Square

Represents a square defined by a single side length.


const s = new Square(5);
s.perimeter(); // 20
s.area();      // 25
  • perimeter() → Returns the perimeter of the square
  • area() → Returns the area of the square

Rectangle

Represents a rectangle defined by width (x) and height (y).


const r = new Rectangle(4, 6);
r.perimeter(); // 20
r.area();      // 24
  • perimeter() → Returns the perimeter of the rectangle
  • area() → Returns the area of the rectangle

Circle

Represents a circle defined by radius (x).


const c = new Circle(7);
c.circumference(); // ~43.98
c.area();          // ~153.94
  • circumference() → Returns the circumference of the circle
  • area() → Returns the area of the circle

Parallelogram

Represents a parallelogram defined by base (x), side (y), and either height or angle.


const p1 = new Parallelogram(8, 5, 4);     // using height
p1.area();      // 32

const p2 = new Parallelogram(8, 5, "60deg"); // using angle
p2.area();      // ~34.641016
  • perimeter() → Returns the perimeter
  • area() → Returns the area (uses either height or angle)

Triangle

Represents a triangle defined by base (x), side (y), and either height or angle.


const t1 = new Triangle(10, 6, 4);        // using height
t1.area();      // 12

const t2 = new Triangle(10, 6, "45deg");  // using angle
t2.area();      // ~21.213203
  • perimeter() → Returns a placeholder (not fully implemented)
  • area() → Returns the area (uses either height or angle)
